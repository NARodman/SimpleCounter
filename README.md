# Android Lab 1 - SimpleCounter

Submitted by: **Nina Lanier**

**SimpleCounter** is an Android app that tracks how many times a user taps a button. Each tap increases the number displayed on the screen by 1.

Time spent: **2 hours** spent in total

## Required Features

The following **required** functionality is completed:

- [x] **User can see a number displayed on the screen** (starts at 0)
- [x] **User can tap on a button to increase the number by 1**
- [x] **Number displayed updates immediately with each tap**

## Optional / Stretch Features Implemented

- [x] Added a **custom background** for the app
- [x] Changed **text color** to white for better visibility
- [x] Changed **button color** to red
